Welcome to OpalUI

A minimalistic CSS framework with theme support, featuring frosted glass and opaque themes with dark/light mode support.

New in v0.12
Layout helpers, tables, alerts, tooltips, spinners, skeletons, progress rings, and much more!

Key Features

Theming

Glass and opaque themes with dark/light modes

Layout Helpers

Stack, cluster, split, grid, and more

Components

Buttons, forms, cards, tables, alerts

Responsive

Mobile-first with breakpoint utilities

Quick Start

npm install @symbion/opalui

Or include via CDN:

<link rel="stylesheet" href="opalui.min.css">
